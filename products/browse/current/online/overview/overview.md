## Browse

Category and collection navigation — when users explore products by browsing categories rather than searching. Includes sorting, filtering, faceted navigation, and within-category ranking.

### Key Components

| Component | Description | Status |
|-----------|-------------|--------|
| Category navigation | Hierarchical category tree traversal | TODO |
| Faceted filtering | Dynamic facets based on category attributes | TODO |
| Within-category ranking | ML-based ordering within a category page | TODO |
| Personalization | Per-user reranking of category results | TODO |

### Differences from Search

- No query — user navigates by clicking categories
- All products in category are candidates (no retrieval step)
- Ranking is the primary ML component
- Facets are pre-computed per category, not per query
