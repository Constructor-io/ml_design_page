## Related Searches

"People also searched for" — query suggestions shown alongside search results to help users refine or pivot their search. Combines co-occurrence mining, semantic similarity, and personalization.

### Key Components

| Component | Description | Status |
|-----------|-------------|--------|
| Co-occurrence mining | Queries frequently issued in same session | TODO |
| Semantic expansion | Queries with similar meaning / intent | TODO |
| Personalized suggestions | User-specific query recommendations | TODO |
| Ranking | Ordering related queries by expected utility | TODO |
| Deduplication | Merging near-duplicate queries | TODO |

### Generation Approaches

- **Session co-occurrence:** queries issued in same session → related
- **Click overlap:** queries leading to clicks on same items → related
- **Embedding similarity:** queries close in semantic space → related
- **LLM generation:** generate reformulations / refinements of the query
