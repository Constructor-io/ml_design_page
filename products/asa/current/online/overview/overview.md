## ASA (Automated Structured Attributes)

Automated extraction and enrichment of product attributes from unstructured catalog data. Powers faceted navigation, filtering, and improves search/recommendations quality.

### Key Components

| Component | Description | Status |
|-----------|-------------|--------|
| Attribute extraction | LLM-based extraction from titles/descriptions | TODO |
| Category classification | Hierarchical product categorization | TODO |
| Value normalization | Standardizing attribute values (colors, sizes) | TODO |
| Quality scoring | Confidence scores for extracted attributes | TODO |
| Cross-customer transfer | Leveraging shared attribute knowledge | TODO |

### Use Cases

- **Faceted navigation:** automatically generate filter facets from extracted attributes
- **Search filtering:** enable attribute-based filtering ("red", "under $50") without manual tagging
- **Recommendation quality:** better item features → better embeddings → better recommendations
- **Catalog health:** detect missing/inconsistent attributes across catalog
