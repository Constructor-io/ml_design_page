## Offsite Discovery

Product discovery outside the e-commerce site — email recommendations, push notifications, ad platform feed optimization. Leverages the same user/item embeddings as on-site but optimized for different contexts and conversion funnels.

### Key Components

| Component | Description | Status |
|-----------|-------------|--------|
| Email recommendations | Personalized product picks for email campaigns | TODO |
| Push notifications | Trigger-based product alerts | TODO |
| Ad feed optimization | Product feed ranking for ad platforms (Google, Meta) | TODO |
| Audience segmentation | User clustering for targeted campaigns | TODO |
| Attribution | Cross-channel conversion tracking | TODO |

### Shared Infrastructure

- User embeddings (same as on-site personalization)
- Item embeddings (same as search/recommendations)
- Behavioral signals (clicks, purchases, browsing history)
- Difference: offline batch scoring, not real-time
